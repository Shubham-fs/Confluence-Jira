# 01 - Project Overview

## Problem

Developer activity is often spread across Jira tickets, Jira workflow history, and Confluence team pages. Manually preparing reports takes time and can miss historical context, especially when tickets have been reassigned or moved across multiple statuses.

## Solution

The Developer Activity Reporting app gives a single dashboard that connects to Confluence Cloud and Jira Cloud. A user selects a team, a member, and an optional date range, then generates activity reports and Excel exports.

## Core Features

### Assigned Issues

Shows Jira issues that were assigned to a member at any point in time. This is not only current assignment; the backend uses Jira JQL history with `assignee WAS "accountId"`.

### Transitions

Shows one-step forward workflow movements from Jira changelog history. With the default workflow, the app detects:

- `To Do -> Build`
- `Build -> Pending QA`
- `Pending QA -> Done`

The original Build -> Pending QA report is still available as a specific transition filter.

### Advanced AI Search

Uses Groq to convert a free-form question into a structured plan and JQL. The dashboard shows the executed JQL, so the user can inspect exactly what Jira search ran.

### Excel Export

Assigned and Transitions reports can be exported as `.xlsx` files. Exports are generated by the backend from the same filters.

## Key Terms

- **JQL**: Jira Query Language.
- **Account ID**: Jira Cloud's stable user identifier.
- **Assignee**: person assigned to an issue at a point in time.
- **Actor**: person who performed a transition.
- **Changelog**: Jira's issue history feed; used to reconstruct status and assignee changes.
- **Workflow statuses**: ordered statuses used to detect one-step forward movement.
- **Executed JQL**: the final project-scoped JQL actually sent to Jira.

## Why This Matters

The app demonstrates practical full-stack integration: a React dashboard, FastAPI backend, Confluence team discovery, Jira reporting, Groq-powered query planning, and auditable JQL execution.
